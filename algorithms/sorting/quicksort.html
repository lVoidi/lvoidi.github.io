Quick Sort

Sorting Medium

Description

QuickSort is a highly efficient, divide-and-conquer sorting algorithm that revolutionized sorting techniques when introduced by Tony Hoare in 1960. Unlike simpler algorithms like Bubble Sort or Selection Sort, QuickSort achieves excellent average-case performance of O(n log n) by intelligently dividing the problem into smaller subproblems.

The algorithm's core principle revolves around selecting a 'pivot' element from the array and partitioning all other elements into two groups: those smaller than the pivot go to the left, and those larger go to the right. This partitioning process continues recursively on each sub-array until the entire array is sorted. What makes QuickSort particularly powerful is its in-place sorting capability, meaning it requires minimal additional memory beyond the input array itself.

Despite its worst-case time complexity of O(n²) when the pivot is consistently poorly chosen (such as always picking the smallest or largest element), QuickSort performs exceptionally well in practice. With proper pivot selection strategies like median-of-three or randomized pivot selection, the worst case becomes extremely rare, making it one of the fastest sorting algorithms for general-purpose use.

How It Works

The QuickSort algorithm follows a systematic divide-and-conquer approach that can be broken down into the following steps:

  1. Choose a Pivot: Select a pivot element from the array. Common strategies include choosing the last element, the first element, the middle element, or using the median-of-three method (picking the median of the first, middle, and last elements). The choice of pivot can significantly impact performance.
  2. Partition the Array: Rearrange the array so that all elements smaller than the pivot come before it, and all elements greater than the pivot come after it. After this partitioning, the pivot is in its final sorted position. This step is crucial and typically uses two pointers: one scanning from the left and one from the right.
  3. Recursively Sort Sub-arrays: Apply the QuickSort algorithm recursively to the sub-array of elements with smaller values (left of the pivot) and the sub-array of elements with greater values (right of the pivot). Each recursive call further divides these sub-arrays until they contain only one element or are empty.
  4. Base Case: The recursion terminates when a sub-array has zero or one element, as such arrays are already sorted by definition. This forms the base case of the recursive algorithm.
  5. Combine Results: Unlike merge sort, QuickSort doesn't require an explicit merge step. Since the partitioning is done in-place, the array becomes sorted naturally as the recursion unwinds.

The efficiency of QuickSort comes from its ability to eliminate multiple elements from consideration with each partition operation, rather than comparing adjacent elements one at a time like simpler sorting algorithms.

def partition(arr, low, high):
    pivot = arr[high]
    i = low - 1
    
    for j in range(low, high):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1

def quicksort(arr, low, high):
    if low < high:
        pi = partition(arr, low, high)
        quicksort(arr, low, pi - 1)
        quicksort(arr, pi + 1, high)

# Helper function to call quicksort
def sort(arr):
    quicksort(arr, 0, len(arr) - 1)
    return arr

#include <iostream>
#include <vector>

class QuickSort {
private:
    static int partition(std::vector<int>& arr, int low, int high) {
        int pivot = arr[high];
        int i = low - 1;
        
        for (int j = low; j < high; j++) {
            if (arr[j] <= pivot) {
                i++;
                std::swap(arr[i], arr[j]);
            }
        }
        
        std::swap(arr[i + 1], arr[high]);
        return i + 1;
    }
    
    static void quicksort(std::vector<int>& arr, int low, int high) {
        if (low < high) {
            int pi = partition(arr, low, high);
            quicksort(arr, low, pi - 1);
            quicksort(arr, pi + 1, high);
        }
    }

public:
    static void sort(std::vector<int>& arr) {
        quicksort(arr, 0, arr.size() - 1);
    }
};

public class QuickSort
{
    private static int Partition(int[] arr, int low, int high)
    {
        int pivot = arr[high];
        int i = low - 1;
        
        for (int j = low; j < high; j++)
        {
            if (arr[j] <= pivot)
            {
                i++;
                (arr[i], arr[j]) = (arr[j], arr[i]);
            }
        }
        
        (arr[i + 1], arr[high]) = (arr[high], arr[i + 1]);
        return i + 1;
    }
    
    private static void QuickSortRecursive(int[] arr, int low, int high)
    {
        if (low < high)
        {
            int pi = Partition(arr, low, high);
            QuickSortRecursive(arr, low, pi - 1);
            QuickSortRecursive(arr, pi + 1, high);
        }
    }
    
    public static void Sort(int[] arr)
    {
        QuickSortRecursive(arr, 0, arr.Length - 1);
    }
    
    // Example usage
    public static void Main(string[] args)
    {
        int[] arr = { 64, 34, 25, 12, 22, 11, 90 };
        
        Console.WriteLine("Original array: " + string.Join(" ", arr));
        Sort(arr);
        Console.WriteLine("Sorted array: " + string.Join(" ", arr));
    }
}
