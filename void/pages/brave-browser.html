Chromium based browsers are better

This was one of the most important lessons i learned around web browsers. Firefox maxis like to say that their browser is better because it is the only one that is different and not based in chromium. However, this is not an advantage at all. It is proven that chromium based browsers are more secure than Firefox. This does NOT mean that they're all more private, however, that is where browsers like Brave come in.

Let's talk about why chromium browsers are more secure. This is because V8 sandbox. Check this and this. Firefox sandbox is way weaker and it makes it easier to exploit. If you think about it, it makes a lot of sense since Mozilla security team is much much smaller than Google one. Also, there is no GPU Sandboxing, and also site isolation is not at the same level as Chromium. It's not even close.

There are private chromium browsers

Firefox fans like to say that chromium browsers are all the same and they all collect your data. This is false, really false. Browsers like Brave might use Chromium, but that does NOT make it the same as, for example, Microsoft Edge.

Even though these browsers use the same engine, Chromium, they are really really different. Even firefox uses chromium parts, see IPC and sandbox.

Why Brave?

Brave Browser is my personal pick from all web browsers. It is not as lightweight as Microsoft Edge, however, it is way more private and secure. Brave Shields is way more secure than using a chromium browser with uBlock Origin. This is because google has done several attempts to make uBO not work. uBlock Origin is well known because it blocks almost all the ads and it is a threat to youtube advertisement, since it blocks it all. That's why uBlock Origin is not that good of an option in chromium browsers, since google wants to get rid of it SO bad. This obviously does not happen in mozilla firefox, since they do not care if the users are blocking youtube advertisement. This is an argument firefox lovers use frequently, however, there is an alternatives. Brave Shields is proven to work as good as uBlock Origin and, even doing some things that uBO can't. For example, brave shields can blocks network requests from extensions, which uBO cannot do as an extension. You can see that tweet for more examples of why Brave is so good.

Browser thing is mostly personal preferences

Something i learned is that browser thing is mostly based in your personal liking. Some people prefer using Microsoft Edge because it integrates really well with Microsoft products. Some other like to use Firefox because it is not chromium. There are many reasons why you would prefer some browsers than others. In my opinion, Brave suits my personal preferences, it is private, secure, fast and is chromium. That is more than enough for me.

Conclusion

Choose the browser you prefer. I told my reasons of why i prefer Brave, but your reasons are mostly valid.
